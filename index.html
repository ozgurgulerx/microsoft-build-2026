How to use this map

The center four blocks are the builder assembly line. Model is the intelligence layer, Context is what the agent can know, Tools are what it can do, and Harness is the production wrapper that gives the agent identity, state, traceability, evaluations, and lifecycle controls.

The horizontal bands explain the surrounding platform. Dev / Backend Builder Substrate covers the developer frameworks, SDKs, and MCP surfaces used to build samples. Productionisation & Governance covers the controls needed before agents touch real users or enterprise data. AI Infrastructure covers the local, cloud, serverless, sandbox, and accelerated compute substrates that make the demos runnable.
