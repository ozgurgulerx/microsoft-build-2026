# Sample 07: Foundry Local Edge Agent

Goal: plan a local-first voice and retrieval agent that can run close to users, devices, or operational data.

## Builder Question

How do I build an agent when data residency, latency, offline operation, or edge execution matters?

## Planned Shape

- Voice or retrieval flow with local inference first and cloud escalation second.

- Offline behavior and cancellation checks.
- Hardware and OS compatibility matrix.
- Cleanup and observability notes for local runs.
