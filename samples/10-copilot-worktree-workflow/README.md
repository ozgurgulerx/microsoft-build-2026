# Sample 10: Copilot Worktree Workflow

Goal: map a parallel-agent development workflow that keeps work isolated, reviewable, and recoverable.

## Builder Question

How do I run multiple coding-agent sessions without losing control of changes, reviews, or rollbacks?

## Planned Shape

- GitHub Copilot app worktree workflow.
- Issue-to-branch-to-PR routing.
- Review gates for generated code, tests, and security notes.
- Conflict and rollback procedure.
- Traceable decision log for each agent session.
- Public demo repo fixture with synthetic code.
