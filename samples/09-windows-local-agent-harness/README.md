# Sample 09: Windows Local-Agent Harness

Goal: define a Windows-local agent harness that uses OS-level containment, identity, policy, and local model controls.

## Builder Question

How do I let a desktop agent act locally without losing enterprise control?

## Planned Shape

- Windows AI local model path.
- MXC-style policy checklist for tool and file access.
- Identity attribution and audit notes.
- Windows 365 Agents variant for managed Cloud PCs.
- Local sandbox, network, and data-boundary checks.
- Security review checklist before any runnable demo.
