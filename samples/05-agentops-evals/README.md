# Sample 05: AgentOps Evals

Goal: create a production-readiness harness for an agent using traces, rubric evaluations, guardrails, optimizer-ready feedback, and ROI proxies.

## Builder Question

How do I know whether an agent is safe, useful, improving, and worth the cost?

## Planned Shape

- Scenario test set.
- Rubric criteria for task success, safety, latency, cost, and tone.
- ASSERT-style adversarial cases where public tooling allows.
- Trace IDs connected to evaluation scores.
- Improvement proposal log with before/after comparison.
- ROI dashboard shape.
