# Sample 02: Foundry IQ Grounding

Goal: demonstrate a retrieval-first agent that grounds answers in managed knowledge sources and reports where its answer came from.

## Builder Question

How do I connect an agent to enterprise and web knowledge without rebuilding a custom RAG platform?

## Planned Shape

- Foundry IQ knowledge base setup notes.
- Agent Framework context provider.
- File, SQL, web, and MCP source variants where public docs allow.
- Source reporting in every answer.
- Evaluation checklist for groundedness and freshness.
