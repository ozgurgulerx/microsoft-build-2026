# Sample 01: Agent Framework + Toolbox

Goal: build a code-first agent with Microsoft Agent Framework and connect it to a governed tool surface through Foundry Toolboxes/MCP.

## Builder Question

How do I give an agent useful tools without wiring every API, auth flow, and lifecycle by hand?

## Planned Shape

- Python or .NET Agent Framework harness.
- One retrieval/context tool.
- One action tool.
- Toolbox/MCP endpoint configuration.
- Trace and tool-call logging.
- Explicit least-privilege notes.
